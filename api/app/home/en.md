## Version Update

#### 0.4.4

1. Fixed a bug where the C# version export template failed to install.
2. Fixed a bug where text overflowed in the homepage announcement bar.
3. Fixed a bug where the engine version could not be switched when importing projects without a version identifier.

1. With the help of Godot Foundation and after multiple negotiations with the Microsoft Store, the app has been relisted.

3. Fixed a bug that caused the Microsoft Store version to fail when importing projects.
4. Removed the built-in store page and replaced it with a forum(only Chinese now).
5. Optimized the download experience for China mirror sources.
