使用须知 ​

网站开源出来的代码，无特殊说明的，遵守MIT协议即可使用。涉及到的教程和文字、以及教程中使用到的代码，依据其作者各自声明的协议进行分发（无声明则原则上禁止使用）。网站里的其他文字内容，未经书面许可禁止使用。

子模块 ​

  • 设置：git submodule add https://atomgit.com/godothub/public public
  • 拉取：git submodule update --init --recursive
  • 更新：cd public && git pull，然后在项目根项目里提交更新的指针信息

Github：git submodule add https://github.com/godothub/godothub.github.io build
